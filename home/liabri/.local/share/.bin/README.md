**ex:** archive extractor, currently supports tar.bz2, tar.xz, bz2, gz, tar, tbz2, tgz, zip, Z, rar, 7z. requires unzip; tar; ...

**exists:** well, does it exists ? currently supports glyph lookup in fonts

**lwien:** print terminal colours

**xlewn:** colour pipette, requires grim; slurp; imagemagick
